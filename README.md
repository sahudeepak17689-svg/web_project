# 🚌 Online Bus Ticket Reservation System

An Online Bus Ticket Reservation System developed using Java Full Stack technologies.  
This project helps users search buses, book tickets, manage reservations, and view booking history easily through an online platform.

---

## 📌 Features

- 🔐 User Registration & Login
- 🚌 Search Available Buses
- 📅 Bus Schedule Management
- 🎫 Online Ticket Booking
- ❌ Ticket Cancellation
- 💳 Payment Management
- 📄 Booking History
- 👨‍💼 Admin Panel for Bus Management
- 📱 Responsive User Interface
